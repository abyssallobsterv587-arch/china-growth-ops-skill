# Vertical Templates

Use these vertical packs to make the skill more commercially useful.

Pick one vertical per run. Do not mix tones unless the user explicitly wants a crossover campaign.

## Beauty / Skincare

Good for:
- Xiaohongshu
- Douyin
- Ecommerce detail pages

Offer angles:
- Fast visible routine improvement
- Texture, convenience, and portability
- Before/after workflow

Avoid:
- Medical claims
- Guaranteed skin outcomes
- Unsupported ingredient claims

Useful outputs:
- Xiaohongshu note title
- Douyin 15-second demo hook
- Product highlight bullets
- Comment reply script for ingredient and usage questions

## Home / Cleaning / Storage

Good for:
- Douyin
- Kuaishou
- Taobao / Pinduoduo

Offer angles:
- Space saving
- Time saving
- Easy transformation demo

Best content:

## Recommended V2 Run Pattern

For better conversions, pair each run with:
- 1 primary platform
- 1 vertical pack
- 1 offer angle
- 1 CTA

This reduces generic copy and makes the output feel tailored.

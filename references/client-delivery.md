# Client Delivery Template

Use this template when the skill is used as part of a paid service.

## Delivery Summary

- Client:
- Date:
- Product:
- Primary platform:
- Secondary platform:
- Offer angle:

## What Was Delivered

- Product scorecard
- Platform plan
- Copy pack
- Browser-ready fields
- Publish checklist
- Review template

## Recommended Next Step

Choose one:
- Publish now
- Revise hook first
- Revise pricing first
- Change platform first
- Change product first

## Copy Pack Included

List exactly what is included:
- Video hook
- Caption
- Listing title
- Benefit bullets
- FAQ
- Chat replies

## Browser-Ready Fields

Provide:
- Product name
- Price
- Description
- Tags
- Link / CTA destination
- Refund language

## Risks

Call out:
- Compliance risks
- Weak trust signals
- Weak CTA
- Weak product-platform fit

## KPI Targets For First Test

- Views target
- Click target
- Inquiry target
- Add-to-cart target
- Conversion signal to watch

## Handoff Rule

Make the handoff explicit:
- What the operator can do automatically
- What still needs human review
- What final publish action the client must confirm

## Reuse Rule

Every delivery should be reusable next week.

Write outputs so they can be:
- revised
- reposted
- adapted to another platform
- compared against results
